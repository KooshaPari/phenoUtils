# PhenoUtils - Utility Workspace

Shared utility crates for the Pheno ecosystem.

## Crates

| Crate | Description |
|-------|-------------|
| `pheno-shell` | CLI shell framework |
| `pheno-fs` | Filesystem utilities |
| `pheno-crypto` | Cryptographic utilities |
| `pheno-net` | Network utilities |
| `pheno-testing` | Testing utilities |

## Usage

```toml
# Cargo.toml
[dependencies]
pheno-shell = { path = "../phenoUtils/crates/pheno-shell" }
pheno-crypto = { path = "../phenoUtils/crates/pheno-crypto" }
```
